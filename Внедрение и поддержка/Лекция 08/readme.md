## 📚 2. Теоретический базис

**BIOS (Basic Input/Output System)** — это базовый набор микропрограмм, записанных в энергонезависимую память (микросхему EEPROM) на материнской плате. BIOS отвечает за старт компьютера, проверку железа и передачу управления загрузчику операционной системы.

При включении ПК запускается процедура самотестирования **POST (Power-On Self-Test)**:
1. Инициализация CPU, чипсета, ОЗУ и видеоадаптера.
2. Проверка контрольных сумм памяти CMOS и заряда батарейки.
3. Стресс-тест оперативной памяти и процессора.
4. Опрос периферии (накопители, мышь, клавиатура) и распределение ресурсов (IRQ).
5. Поиск загрузчика ОС согласно установленному приоритету (Boot Priority).

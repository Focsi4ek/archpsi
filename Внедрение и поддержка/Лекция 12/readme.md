**Цель работы:** Освоить методы автоматизированного сбора информации о компьютерной системе, научиться анализировать отчеты аудита и проводить инвентаризацию ресурсов ПК.

**Используемый инструментарий:**
Для проведения аудита использовалась утилита **WinAudit v3.3** (by PX Server). Это портативное, бесплатное решение для Windows, не требующее установки. Она выполняет глубокое сканирование реестра, WMI, файловой системы и возвращает структурированный отчет по сотням параметров.

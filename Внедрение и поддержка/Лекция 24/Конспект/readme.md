# Конспект: Операционная система Linux

## 1. Что такое Linux

**Linux** — общее название Unix-подобных операционных систем, построенных на основе одноимённого ядра и собранных для него библиотек и системных программ.

**Создатель:** Линус Торвальдс, студент из Хельсинки.

Linux — достаточно простая и прозрачная операционная система. Высокий уровень защиты привлекает к ней пользователей, хотя требует изучения непривычных терминов, программ и способов работы.

**Открытость и свобода:**
- Ядро, основные компоненты системы и большинство пользовательских приложений — свободно распространяемые программы
- Исходный код открыт: каждый может его скачать, изучить, модифицировать и переиздать
- В отличие от Windows (закрытый код, лицензия запрещает копирование, передачу другим людям, использование на нескольких устройствах), Linux можно свободно копировать и распространять

## 2. Дистрибутивы Linux

Дистрибутив — это сборка ядра Linux вместе с набором программ и собственной программой установки.

**Особенности дистрибутивов:**
- У каждого своя программа установки — позволяет обычному пользователю получить рабочую систему без глубокого понимания процесса
- Назначение программы установки во всех дистрибутивах одинаковое, реализация — разная
- Дистрибутивы отличаются друг от друга набором предустановленных программ и настройками по умолчанию

**Ubuntu** — самый популярный дистрибутив Linux, со слоганом «Linux для людей».

## 3. Файловая система Linux

### Единое дерево каталогов

В Linux **корневая папка только одна** (`/`), все остальные папки вложены в неё. Для пользователя файловая система выглядит как единое целое, даже если физически данные хранятся на разных устройствах/разделах.

### Монтирование

Чтобы получить такое единое дерево с одним корнем, используется **монтирование**.

| Термин | Определение |
|---|---|
| Монтирование | Подключение файловой системы, расположенной на другом устройстве, в одну из папок единого дерева |
| Точка монтирования | Пустая папка, в которую подключается (монтируется) файловая система |
| Размонтирование | Отключение ранее смонтированной файловой системы; папка-точка монтирования снова становится пустой |

**Автоматизация:** пользователю обычно не нужно монтировать/размонтировать вручную — это происходит автоматически:
- при загрузке системы монтируются все устройства с частями файловой системы;
- при остановке системы все они размонтируются.

### Файловая система ext3

В Linux чаще всего используется **журналируемая файловая система ext3**, где размер кластера может составлять от 1 до 8 Кбайт.

> Журналируемая файловая система ведёт журнал изменений, что повышает устойчивость к сбоям (например, при внезапном отключении питания) по сравнению с не журналируемыми ФС.

## 4. Преимущества Linux по сравнению с Windows

| Критерий | Windows | Linux |
|---|---|---|
| **Вирусы и безопасность** | Подавляющее большинство вредоносных программ атакуют именно Windows | Существует около десятка вирусов под Linux, но они не способны поразить систему в целом |
| **Надёжность** | Низкая надёжность: частые сбои, потеря данных | Зарекомендовала себя как надёжная система — может работать бесперебойно годами без переустановки |
| **Стоимость** | Высокая стоимость лицензий | Бесплатная операционная система |
| **Системные требования** | Значительно возрастают с каждой новой версией | — |
| **Обновления и исходный код** | Официальные обновления доступны с серверов Microsoft | Регулярно выходят новые версии ПО, есть доступ к исходным кодам |
| **Исходный код** | Закрытый | Открытый |
| **Лицензия** | Запрещено копировать, передавать другим, использовать на нескольких компьютерах | Свободная: можно копировать, скачивать исходный код, модифицировать и переиздавать |

## Краткое резюме

- Linux — открытая, бесплатная, Unix-подобная ОС, созданная Линусом Торвальдсом
- Существует множество дистрибутивов (например, Ubuntu) с разным набором ПО, но единым принципом установки
- Файловая система Linux имеет единый корень `/`; разные устройства подключаются к дереву через механизм монтирования
- Чаще всего используется журналируемая ФС ext3
- Главные преимущества перед Windows: безопасность, надёжность, бесплатность, открытость исходного кода
